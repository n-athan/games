Rules

  • Try to paint all the squares the right color.
  • The buttons set the paint color.
  • The first try is 'free', there is no paint selected yet.
  • You get a few extra 'free' tries. Be mindful when to use them.
  • If you paint a square the wrong color, you will lose.
  • The number on a square indicates the number of adjacent tiles with the same color (left, right, top, bottom and diagonal).
  • All tiles of the same color are touching, this may be diagonal.
