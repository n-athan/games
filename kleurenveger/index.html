Rules

  • Try to paint all the squares the right color.
  • The buttons set the ink color.
  • The first move is 'free', there is no ink selected yet.
  • You get a few extra 'free' tries. Be mindful when to use them.
  • If you paint a square the wrong color, you will lose.
  • The number on a square indicates the number of adjacent squares with the same color (left, right, top, bottom and diagonal).
  • A square is always adjecent to a square with the same color, this may be diagonal.

Uitleg

  • Probeer alle vakjes de juiste kleur te verfen.
  • Met de knoppen kun je de kleur inkt kiezen.
  • De eerste zet is vrij, er is nog geen inkt geselecteerd.
  • Je krijgt een paar extra vrije zetten. Bedenk goed wanneer je die wilt gebruiken!
  • Als je een vakje de verkeerde kleur verft, dan heb je verloren.
  • Het getal op een vakje laat zien hoeveel aangrensende vakjes dezelfde kleur hebben. Vakjes kunnen in alle richtingen aangrensend zijn. (links, rechts, boven, onder en diagonaal).
  • Een vakje raakt altijd een ander vakje van dezelfde kleur, dit kan ook diagonaal zijn.
